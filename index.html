How to install Keras on Jetson Nano.

1) install Python 3.

sudo apt-get install python3.6

2) install TensorFlow. 

https://devtalk.nvidia.com/default/topic/1048776/official-tensorflow-for-jetson-nano-/

  1. sudo apt-get install python3-pip libhdf5-serial-dev hdf5-tools
  2. pip3 install --extra-index-url https://developer.download.nvidia.com/compute/redist/jp/v42 tensorflow-gpu==1.13.1+nv19.5 --user

3) install Keras.

https://keras.io/#installation

sudo pip3 install keras

Note: if the installation fails due to lacking of library, please install the libary first.
